# 🍌 Banana

A experimental project that stores files inside podcast episodes.

It works by converting binary data into audio tones (similar to old dial-up modems), uploading the audio as a podcast episode, and later decoding the episode back into the original file.

## In other words:
Use Spotify podcasts as a weird file storage system.

## Concept
file → binary → audio tones → mp3 → podcast podcast → mp3 → audio decode → binary → file

## Features (planned)
- Encode files into audio
- Upload encoded audio as podcast episodes
- Download podcast episodes
- Decode audio back into files
